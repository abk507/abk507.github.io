Áudio do motivo para usar git

        O Git é uma ferramenta para simplificar controle de versões e a colaboração em projetos.
        É perfeito para se anotar alterações feitas em um projeto e poder testar ideias sem medo de quebrar o código e não poder voltar atrás.
        Ele ainda permite facilidade no trabalho em conjunto com a feature de branches, que evita a sobreposição de trabalho de outras pessoas
        sobre o seu, ainda possibilitando desenvolver recursos independentemente uns dos outros.
        E quando estiver satisfeito com as alterações, você pode juntá-las de volta á branch principal.
        
        O Git também oferece um histórico detalhado de todas as alterações feitas em seu projeto.
        Isso significa que você pode voltar às versões anteriores do código, comparar alterações e identificar quando, por quem
        e por que cada modificação foi feita, já que permite nomear os commits.

        No geral, é uma ferramenta muito útil para diversas profissões, e indispensável para programadores.
